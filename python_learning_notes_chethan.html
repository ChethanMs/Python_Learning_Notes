This document summarizes the Python concepts learned so far with simple examples.

1. Variables

Variables store data values.

name = "Chethan" age = 29 salary = 50000 print(name, age, salary)

2. Data Types

Common Python data types:

  • int → 10
  • float → 3.14
  • string → "Python"
  • bool → True / False
x = 10 y = 3.5 name = "Python" flag = True

3. Type Conversion

age = int(input("Enter age: ")) price = float(input("Enter price: ")) text = str(100)

4. Arithmetic Operators

a = 10 b = 3 print(a + b) print(a - b) print(a * b) print(a / b) print(a % b) print(a ** b)

5. Input and Output

name = input("Enter your name: ") print("Hello", name)

6. abs() Function

Returns absolute value.

num = int(input("Enter number: ")) print(abs(num))

7. String Concatenation

first = "Chethan" last = "Kumar" full = first + " " + last print(full)

8. String Repetition

print("Python " * 3)

9. String Length

name = "Python" print(len(name))

10. String Indexing

word = "Python" print(word[0]) print(word[-1])

11. Escape Sequences

print("Hello\nWorld") print("Python\tProgramming")

12. String Formatting (f-strings)

name = "Chethan" age = 29 print(f"My name is {name} and I am {age} years old")

13. Comments

# This is a comment print("Python learning")

14. Comparison & Logical Operators

a = 10 b = 20 print(a > b) print(a < b) if a < b and b > 15: print("Condition true")

15. Membership Operator

word = "python" if "py" in word: print("Substring found")

16. Bitwise Operators (Basic)

a = 5 b = 3 print(a & b) print(a | b) print(a ^ b)
