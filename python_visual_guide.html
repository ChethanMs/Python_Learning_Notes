Strings
str methods
Immutable sequences of characters. Rich built-in method library for manipulation.
strings.py

print(s[0])         # H (indexing)
print(s[0:5])       # Hello (slice)
print(s.upper())    # HELLO, PYTHON!
print(s.lower())    # hello, python!
print(s.replace("Hello", "Hi"))
print(s.split(", "))
print(len(s))       # 14

if / elif / else
branching
Branching based on conditions. Python uses indentation (4 spaces) to define blocks.
if
First condition → runs if True
elif
Additional check if previous failed
else
Fallback — all others failed
ifelse.py
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 70:
    print("Grade: B")
elif score >= 50:
    print("Grade: C")
else:
    print("Grade: F")
▶ Output
Grade: B
🛑
Break & Continue
loop control
break exits the loop entirely. continue skips to the next iteration.
break
① → ② → ③ →
■ STOP
continue
① → ② →
⤴ SKIP → ④
break.py
for i in range(10):
    if i == 5:
        break        # stop at 5
    print(i, end=" ")

print()
for i in range(6):
    if i == 3:
        continue     # skip 3
    print(i, end=" ")
▶ Output
0 1 2 3 4
0 1 2 4 5

iteration
Iterates over any sequence — range, list, string, dict, etc.

for i in range(5):

While Loop
condition-based
Repeats as long as the condition is True. Be careful of infinite loops!
i = 0
↓
i < 5?
True ✓
print(i)
i += 1
↺
False ✗
■ EXIT
while.py
i = 0
while i < 5:
    print(i, end=" ")
    i += 1
▶ Output
0 1 2 3 4
⬛
Nested Loops
loop in loop
A loop inside another loop. The inner loop runs fully for every iteration of the outer loop.
Outer(i) × Inner(j) = 3×3 grid:
0,0
0,1
0,2
1,0
1,1
1,2
2,0
2,1
2,2
nested.py
for i in range(3):       # outer
    for j in range(3):   # inner
        print(f"{i},{j}", end=" ")
    print()  # newline after inner
▶ Output
0,0 0,1 0,2
1,0 1,1 1,2
2,0 2,1 2,2
